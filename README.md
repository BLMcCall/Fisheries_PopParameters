# Fisheries_PopParameters
February 2020

Population parameters of non-game species derived from length frequency data in TropFishR

* 'DataAnalysis.R' has R code for pop parameters of the Paleback Darter
* 'lfq.csv' has the length frequency data for the species-level analysis
* 'caddo_lfq.csv' has the length frequency data for the Caddo River population of Paleback Darters
* 'ouachita_lfq.csv' has the length frequency data for the Ouachita River population of Paleback Darters
* 'CPUE_ttest.R' has the seasonal comparisons of sex ratios and catch per unit efforts between populations using paired t-tests and anovas where appropriate
